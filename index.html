Instructions:

Typing Method: Type phonetic equivalents (Latin letters) and they will automatically convert to Amharic characters as you type. Pressing space, punctuation, or enter finalizes the current character.

Examples:

  • s + e = ሰ
  • s + u = ሱ
  • s + i = ሲ
  • s + a = ሳ
  • s + ee = ሴ (or s + e + e)
  • s + e (then space/punctuation) = ስ
  • s + o = ሶ
  • sh + a = ሻ
  • t + t + a = ጣ

Special Cases:

  • For the 6th form (e.g., ስ, ል, ም), type the consonant followed by 'e' and then press space or punctuation (e.g., 's' + 'e' + ' ' -> ስ). Alternatively, typing just the consonant might work in some contexts (e.g., 's' followed by another consonant).
  • Some sounds require multiple letters (e.g., 'sh', 'ch', 'ny', 'zh', 'ts', 'gn').
  • Vowels like 'a', 'e', 'i', 'o', 'u' alone map to አ, እ, ኢ, ኦ, ኡ respectively.

You can also click on the Amharic characters or keys above to insert them directly.
